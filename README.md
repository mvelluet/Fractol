# Fractol

## Usage

./fdf Mandelbrot

## Key

Zoom in  	: Click left or scroll in

Zoom out 	: Click right or scroll out

up			: ↑

down		: ↓

left		: ←

right		: →

color rgb	: > or < 

color red	: r

color blue	: b

color green	: g
